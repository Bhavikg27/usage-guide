# Astroji Usage Guide - React App

This project has been migrated from HTML/CSS to React + Vite.

## Getting Started

1.  Open this folder in your terminal:
    ```bash
    cd astroji-guide
    ```

2.  Install dependencies (if not already done):
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Build for production:
    ```bash
    npm run build
    ```

## Features

-   **Modern Stack**: React 18, Vite, Framer Motion.
-   **Componentized**: Modular UI (Hero, Navbar, ProductCard, etc.).
-   **Interactive**: 3D Tilt effects, Language Toggling (En/Hi), Real-time Search.
-   **Performance**: Optimized build and assets.
